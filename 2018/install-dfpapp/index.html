正文

话说 Deepin 团队已经使用 Flatpak 封装自家应用，这无疑极大方便了其他发行版用户。但是 Deepin 没有直接提供 Flatpak 源等等，所以想要安装还是要自己动手。

准备工作

  • 好一点的网络
  • 安装了 Flatpak 的 Linux 发行版 (x86_64)
  • 洗把脸

安装 Deepin Runtime

Flatpak 应用运行需要一套运行时（Runtime）作为基础环境，Freedesktop、GNOME、KDE 都做了一套供 GTK/Qt 等应用开发者选用。Deepin 则自己做了一套。

下载 com.deepin.platform-15.5_1.0.4+r2_amd64.deb deepin-fprt-org.deepin.flatdeb.base.platform-15.5_1.0.5+r7_amd64.deb，解压出打包好的 bundle 文件 （版本号可能不同）

（以下均以 TUNA 镜像源为例，官方源实在无法直视）

然后安装

flatpak install bundle --bundle

安装应用

Deepin 打包好的应用包括：

  • 深度日历
  • 深度计算器
  • 深度看图
  • 深度影院
  • 深度音乐
  • 深度录屏
  • 深度录音
  • 深度截图
  • 还有好多……
  • 我列不完了……
  • Shadowsocks-Qt5 （？？？）

https://mirrors.tuna.tsinghua.edu.cn/deepin/pool/main/d/

Ctrl + F deepin-fpapp

这里以「深度日历」为例

下载 deepin-fpapp-org.deepin.flatdeb.deepin-calendar_1.2.3+r1_amd64.deb

依照上文，解压出 bundle，如法炮制

flatpak install bundle --bundle
Installing: org.deepin.flatdeb.deepin-calendar/x86_64/master from bundle bundle
Now at 38c791026f75.

org.deepin.flatdeb.deepin-calendar 是应用包名

运行

安装完的 Flatpak 应用可以通过命令行运行

flatpak run org.deepin.flatdeb.deepin-calendar

每次都通过命令启动肯定会让人疯掉，显然需要写一个 .desktop 文件，或者用类似 KDE Menu Editor 这样的工具来新建一个

[Desktop Entry]
Exec=flatpak run org.deepin.flatdeb.deepin-calendar
...
